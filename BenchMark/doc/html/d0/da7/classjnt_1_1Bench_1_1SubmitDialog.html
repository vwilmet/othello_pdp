Description détaillée

SubmitDialog provides a Dialog box for filling in a benchmark submission. It extracts the relevent information from System.Properties of the VM, and provides text boxes for filling in other information. It uses Bench.SendMail to send the message, and uses callbacks to the Applet (if any) to inform of the success or failure of the submission.
