\subsection{Description détaillée}
Provides a stopwatch to measure elapsed time. 
\begin{DoxyDescription}
\item[{\bfseries Example of use\-:} ]
\begin{DoxyPre}

      Q.start();
      //
      // code to be timed here ...
      //
      Q.stop();
      System.out.println("elapsed time was: " + Q.read() + " seconds.");
