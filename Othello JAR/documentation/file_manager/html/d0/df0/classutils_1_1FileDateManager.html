static String utils.FileDateManager.getDateFormatAAAAMMJJHHMMSS ( )
static

Méthode retournant la date sous format AAAA-MM-DD-HH-MM-SS avec :

  • AAAA : Année -> 2014
  • MM : Mois -> 01
  • DD : jour -> 12
  • HH : heure -> 24
  • MM : minute -> 36
  • SS : seconde -> 38
  • Exemple : 2014-02-16-19-27-38

la date + heure en String
