Description détaillée

Interface de gestion des différentes intelligences artificielles. Cette interface contient les méthodes afin d'utiliser les différentes intelligences artificielles. Il est important d'utiliser la méthode notifyChosenMove com.ai.ArtificialIntelligence#notifyChosenMove afin de signaler à l'IA le coup choisit afin qu'elle positionne automatique la sentinelle au bon endroit dans l'arbre. L'arbre permet de prévoir les coups pour les 2 joueurs, il faut donc notifier à la fois le coup choisi par le joueur 1, mais aussi le joueur 2, qu'il soit un joueur réel ou simplement l'ordinateur.
