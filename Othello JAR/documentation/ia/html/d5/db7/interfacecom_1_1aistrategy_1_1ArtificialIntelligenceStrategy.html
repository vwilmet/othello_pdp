Description détaillée

Interface permettant d'implémenter les différentes stratégies/intelligences artificielles Cette interface contient les méthodes afin d'utiliser les différentes intelligences artificielles. Par exemple : la fonction nextMove permet de retourner le prochain mouvement calculer par l'IA...
