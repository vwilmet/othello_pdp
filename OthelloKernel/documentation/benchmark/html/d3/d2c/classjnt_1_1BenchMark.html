jnt.BenchMark.BenchMark ( BenchMarkResultEvent  event)

Constructeur qui nécessite l'utilisation de l'interface afin de communiquer les résultats du calcul Attention : Vous devez fournir une référence valide pour l'objet en paramètre sinon il sera impossible de connaître l'avancement du calcul général et de récupèrer les valeurs finales

Paramètres
eventL'interface de communication avec le BenchMark

Documentation des fonctions membres

void jnt.BenchMark.launch ( )

Méthode à appeler afin de lancer le BenchMarking
Cette méthode renvoie le résultat à travers l'objet passé en paramètre de la méthode jnt.BenchMarkResultEvent#onEnd(BenchMarkResult) de l'interface jnt.BenchMarkResultEvent
Si l'interface de communication n'est pas valide alors le BenchMarking ne se lance pas et s'arrète immédiatement
